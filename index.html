Privacy Policy for iMagic Consult

iMagic Consult is a Chrome extension used to help clinicians format consultation notes from supported hospital web pages.

Data Collection

The extension may read visible patient information from the supported RME website only when the user actively uses features such as consultation generation, CPPT formatting, selector testing, or template injection.

Data Storage

Configuration, templates, session data, and temporary CPPT metadata may be stored locally in the browser using Chrome storage. If cloud sync is enabled, configuration data may be synchronized through Supabase.

Data Sharing

The extension does not sell user data. The extension does not share patient data with third parties except when the user-configured Supabase sync feature is enabled.

Remote Code

The extension does not execute remote JavaScript code.
